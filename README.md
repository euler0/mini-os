# mini-os

Sometimes I build my own operating system, one pebble at a time.

## Prerequisites

1. GNU binutils (i686-elf target)
2. QEMU
3. BSD Make
4. mkdep
5. NASM
6. Clang

In macOS,

```sh
$ brew install i686-elf-binutils
$ brew install qemu
$ brew install bsdmake
$ brew install nasm
```
